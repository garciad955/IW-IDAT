Open Source


We seek to provide an intuitive tool so that users from various areas have the possibility of using it. On the other hand, we provide access to the application code so that users have the ability to integrate their own functionalities according to their needs.

What can we do?


Based on the CRISP-DM methodology, it is possible through the tool to carry out the various phases of said methodology.

  • Business Undestanding
  • Data Understanding
  • Data Preparation
  • Modeling
  • Deployment

Modeling techniques


The tool has several types of modeling, such as:

  • Artificial Neural Network
  • Decision trees
  • Regression algorithms
  • K nearest neighbors algorithm

Designed for the user to determine the most appropriate model according to the problem he is working on.
